Didaskalos

Frequency-based Ancient Greek grammar textbook builder.

Didaskalos builds a grammar textbook from treebanks and modular lessons. It can use sample XML treebanks or your own uploads, then ranks grammatical structures by frequency to focus the lessons on what appears most in the texts.

  • Upload treebanks or use the bundled samples
  • Generate case-based or declension-based lessons
  • Download the result as Markdown or HTML

دیداسکالوس

سازندهٔ کتاب‌درسی دستور یونانیِ باستان بر پایهٔ فراوانی.

دیداسکالوس از درخت‌بانک‌ها و درس‌های ماژولار برای ساخت یک کتاب دستور استفاده می‌کند. می‌توانید از فایل‌های XML نمونه یا درخت‌بانک خودتان استفاده کنید و سپس ساختارهای دستوری را بر اساس فراوانی مرتب کنید تا درس‌ها روی الگوهای پرتکرار تمرکز کنند.

  • درخت‌بانک‌های نمونه یا فایل‌های خودتان را بارگذاری کنید
  • درس‌های مبتنی بر حالت یا صرف را تولید کنید
  • خروجی را به صورت Markdown یا HTML دانلود کنید

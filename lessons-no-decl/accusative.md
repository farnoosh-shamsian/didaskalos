## Introduction to the Accusative Case

The accusative in Ancient Greek marks the direct object of a verb. Its shape is obtained by adding a case‑ending to the noun's stem, and the ending that is used depends on the declension to which the noun belongs.

---

### 1. First‑declension (‑α / ‑η stems)

|                      | Singular             | Plural   |
| -------------------- | -------------------- | -------- |
| Masculine / Feminine | **‑αν** (or **‑ην**) | **‑ας** |

The accusative replaces the nominative ‑α/‑η (sg.) and ‑αι (pl.).

_Example_ – **ἀγορά** "market" → **ἀγοράν** (sg. "market" as object) → **ἀγοράς** (pl. "markets" as objects).
_Example_ – **τιμή** "honour" → **τιμήν** → **τιμάς**.

---

### 2. Second‑declension (‑ο stems)

|          | Masculine / Feminine | Neuter |
| -------- | ------------------ | -------- |
| Singular | **‑ον**            | **‑ον** (= nominative) |
| Plural   | **‑ους**           | **‑α** (= nominative) |

For neuter nouns the accusative coincides with the nominative (‑ον in the singular, ‑α in the plural). Feminine nouns of this declension (such as ὁδός) decline exactly like the masculines.

_Examples_

- Masculine: **ὁ φίλος** "the friend" → **τὸν φίλον** (sg.) → **τοὺς φίλους** (pl.).
- Feminine: **ἡ ὁδός** "the road" → **τὴν ὁδόν** (sg.) → **τὰς ὁδούς** (pl.).
- Neuter: **τὸ δῶρον** "the gift" → **τὸ δῶρον** (sg.) → **τὰ δῶρα** (pl.) – the forms are identical to the nominative.

---

### 3. Third‑declension (consonant and vowel stems)

|          | Masculine / Feminine | Neuter |
| -------- | -------------------- | ------ |
| Singular | **‑α** (vowel stems: **‑ν**) | = nominative |
| Plural   | **‑ας** (ι-stems: **‑εις**) | **‑α** |

The ending is attached directly to the stem; no vowel is inserted.

_Examples_

- Masculine: **φύλαξ** "guard" → **φύλακα** (sg.) → **φύλακας** (pl.).
- Feminine: **πόλις** "city" (ι-stem) → **πόλιν** (sg.) → **πόλεις** (pl.).
- Neuter: **σῶμα** "body" → **σῶμα** (sg. = nominative) → **σώματα** (pl.).

---

### How the morphology is built

1. **Identify the declension** of the noun (first, second, or third).
2. **Strip the nominative ending** to obtain the stem.
3. **Attach the appropriate accusative suffix** listed above for the noun's gender, number, and declension.

These suffixes give each noun its accusative form, allowing it to serve as the direct object in a sentence.

# Lesson: The Present Optative Middle/Passive of Deponent Verbs

*Target level: Intermediate*
*Prerequisites: Deponent verbs, present indicative middle/passive, optative mood basics*

---

## 1. What This Form Is

Deponent verbs (citation form in **-μαι**) use middle/passive endings with active meaning. Their **present optative** expresses an ongoing or general wish or possibility: βουλοίμην ἄν "I would wish", εἰ δύναιτο "if he were able".

The heavy hitters in this category are exactly the deponents you use most:
**βούλομαι** "want", **δύναμαι** "be able", **μάχομαι** "fight", **ἕπομαι** "follow", **ἐπίσταμαι** "know how".

## 2. Morphology

### 2.1 Thematic deponents — -οίμην, -οιο, -οιτο

**βούλομαι "want"**:

| Person | Form | Gloss |
|---|---|---|
| 1 sg | **βουλοίμην** | may I want / I would want |
| 2 sg | **βούλοιο** | |
| 3 sg | **βούλοιτο** | |
| 1 pl | **βουλοίμεθα** | |
| 2 pl | **βούλοισθε** | |
| 3 pl | **βούλοιντο** | |

Same pattern: μάχομαι → **μαχοίμην, μάχοιτο**; ἕπομαι → **ἑποίμην, ἕποιτο**; γίγνομαι (present stem) → **γιγνοίμην, γίγνοιτο**.

### 2.2 Athematic deponents — -αίμην, -αιο, -αιτο

Deponents whose stem ends in -α (δύναμαι, ἐπίσταμαι, πρίαμαι) attach the optative suffix directly to the stem vowel:

**δύναμαι "be able"**:

| Person | Form |
|---|---|
| 1 sg | **δυναίμην** |
| 2 sg | **δύναιο** |
| 3 sg | **δύναιτο** |
| 1 pl | **δυναίμεθα** |
| 2 pl | **δύναισθε** |
| 3 pl | **δύναιντο** |

Same pattern: ἐπίσταμαι → **ἐπισταίμην, ἐπίσταιτο**; πρίαμαι "buy" → **πριαίμην** (technically an aorist, used as the optative of ὠνέομαι).

## 3. Key Rules & Common Exceptions

| Rule | Explanation | Example |
|---|---|---|
| **1. Thematic verbs take -οι-, athematic take -αι-** | βουλοίμην but δυναίμην. If the indicative is βούλομαι (with -ο-), the optative has -οι-; if δύναμαι (no theme vowel), -αι-. | |
| **2. 2 sg loses σ** | βούλοι(σ)ο → βούλοιο; δύναι(σ)ο → δύναιο. | |
| **3. One form, middle or passive** | For deponents the question does not arise — the meaning is active. For semi-deponents context decides. | |
| **4. Epic/Ionic 3 pl -οίατο / -αίατο** | Homer and Herodotus: μαχεοίατο, δυναίατο. | |
| **5. Accent stays on the optative diphthong where possible** | δυναίμην, ἐπισταίμην — final -αι of the optative counts long for accent. | |

## 4. Example Sentences

- βουλοίμην ἂν ὑμῖν φίλος εἶναι. — "I would like to be your friend." (polite potential)
- εἰ δύναιτο, βοηθήσειεν ἄν. — "If he were able, he would help."
- ἤρετο εἰ ἕποιντο οἱ σύμμαχοι. — "He asked whether the allies were following." (optative in indirect question, past sequence)
- οὐκ ἂν μαχοίμην τοῖς θεοῖς. — "I would not fight against the gods."
- εἴθε ἐπίσταιο ὅσα κακὰ ποιεῖς. — "If only you knew how much harm you are doing!"

## 5. Usage & Context

1. **βουλοίμην ἄν** is the standard Attic courtesy formula — "I should like ..." — softer than βούλομαι. Very common in Plato's dialogues.
2. **General/repeated action in the past:** εἰ (ὁπότε) + optative in the subordinate clause with imperfect in the main clause: ὁπότε μάχεσθαι δύναιντο, ἐμάχοντο "whenever they were able to fight, they fought".
3. **Indirect discourse in past sequence:** present indicatives of the original speech become present optatives.
4. Distinguish the **present** (ongoing: "be able, keep fighting") from the **aorist** optative of the same verbs (single act: γένοιτο, μαχέσαιτο).

# Lesson: The Aorist Active Participle – Irregular Forms  

*Target level: Beginner‑Intermediate*  
*Prerequisites: Greek alphabet, basic verb endings, the concept of aorist tense.*

---

## 1. What is the Aorist Active Participle?

| Term | Meaning in Greek | Rough English equivalent |
|------|------------------|--------------------------|
| **Aorist** | Simple, “punctual” past (or sometimes future) action, without reference to duration. | “did / have done” |
| **Active** | The subject of the verb performs the action. | “he/she/it **does** …” |
| **Participle** | A verbal adjective that can be declined like a noun. | “the one who …ed”, “having …ed”, “while …ing” |

The **aorist active participle** therefore describes a *completed* action performed by the subject, and it behaves like an adjective (it has gender, number, case).  

> **Example** – *ἐλθών* “having come / the one who came”.

---

## 2. Forming the Aorist Active Participle  

### 2.1 Regular pattern  

1. **Find the aorist active stem** (the stem used for the aorist indicative).  
2. **Add the participial suffix** **‑ων / ‑οῦσα / ‑ον** (masc./fem./neut. sg.) and the corresponding plural endings **‑όντες / ‑οῦσαι / ‑όντα**.  

| Gender / Number | Ending (singular) | Ending (plural) |
|-----------------|-------------------|-----------------|
| Masculine       | ‑ **ων**          | ‑ **όντες**    |
| Feminine        | ‑ **οῦσα**        | ‑ **οῦσαι**    |
| Neuter          | ‑ **ον**          | ‑ **όντα**     |

**Accent**: The acute accent falls on the ending (‑ **ῶν**, ‑ **οῦσα**, ‑ **όν**, etc.).

### 2.2 Irregular (suppletive) aorist stems  

Many common verbs have **irregular aorist stems** that differ from the present stem. The participle is built on those irregular stems, not on the present stem.

Below are the most frequently encountered irregular verbs and their aorist active participles.

| Verb (present) | Aorist indicative (active) | Aorist stem | Active Participle (masc. nom. sg.) |
|----------------|----------------------------|-------------|-----------------------------------|
| **ἔρχομαι** “to come” | ἦλθον | **ἐλθ‑** | **ἐλθών** |
| **δίδωμι** “to give” | ἔδωκα | **δω‑** | **δῶν** |
| **γίγνομαι** “to become” | ἐγένοντο | **γεν‑** | **ἐγεννώς** |
| **φημί** “to say” | ἔφησα | **φησ‑** | **φησῶν** |
| **πίνω** “to drink” | ἔπιον (or ἔπιον) – regular, shown for contrast | **πιο‑** | **πιον** (regular) |
| **ἵστημι** “to set, stand” | ἔστησα | **στησ‑** | **ἑστηκώς** (perfect‑like, see note) |
| **βάλλω** “to throw” | ἔβαλον | **βαλ‑** | **βαλῶν** (regular) |

> **Note:** Some verbs (e.g., ἵστημι) have a *perfect‑like* aorist participle because the aorist stem itself is formed from a perfect stem. For the purpose of this lesson we treat them as irregular aorist participles.

### 2.3 Full paradigm (example: **δῶν** from *δίδωμι*)

| Case | Masculine | Feminine | Neuter |
|------|-----------|----------|--------|
| **Singular** | | | |
| Nominative | **δῶν** | **δούσα** | **δόν** |
| Genitive   | **δόντος** | **δούσης** | **δόντος** |
| Dative     | **δόντι** | **δούσῃ** | **δόντι** |
| Accusative | **δόντα** | **δούσαν** | **δόν** |
| Vocative   | **δῶν** | **δούσα** | **δόν** |
| **Plural** | | | |
| Nominative | **δόντες** | **δούσαι** | **δόντα** |
| Genitive   | **δόντων** | **δούσων** | **δόντων** |
| Dative     | **δόνσι(ν)** | **δούσαις** | **δόνσι(ν)** |
| Accusative | **δόντας** | **δούσας** | **δόντα** |
| Vocative   | **δόντες** | **δούσαι** | **δόντα** |

The same pattern of endings applies to all aorist active participles; only the **stem** changes.

## 3. Key Rules & Common Exceptions  

| Rule | Explanation | Example |
|------|-------------|---------|
| **1. Use the aorist stem** | The participle never uses the present stem. | *ἔρχομαι* → stem **ἐλθ‑**, not **ἔρχ‑**. |
| **2. Add the regular participial endings** | Even irregular stems take the same endings **‑ων / ‑οῦσα / ‑ον**. | **ἐλθ‑** + **‑ών** = **ἐλθών**. |
| **3. Accent on the ending** | The acute accent is placed on the final syllable of the ending. | **δῶν**, **δούσα**, **δόν**. |
| **4. Contracted stems** | If the aorist stem ends in a vowel, contraction may occur before the suffix. | *φησ‑* + *‑ων* → **φησῶν** (no extra vowel). |
| **5. Suppletion** | Some verbs have completely different stems (suppletive). Memorise them. | *ἔρχομαι* → **ἐλθ‑**. |
| **6. Verbs ending in ‑μι** | The aorist participle often uses the same stem as the aorist indicative (e.g., *ἱστῶ* → **ἑστηκώς**). | *ἵστημι* → **ἑστηκώς**. |
| **7. No aorist participle for *εἰμί*** | The verb “to be” lacks an aorist, so there is no aorist active participle. | — |

## 4. Example Sentences  

| Greek (with participle) | Gloss | Translation |
|--------------------------|-------|-------------|
| **Ἐλθὼν** εἰς τὴν πόλιν, ὁ ἄνθρωπος ἀπῆλθεν. | *Elthōn* eis tēn polin, ho anthrōpos apēlthen. | **Having come** into the city, the man left. |
| **Δῶν** τὸ δῶρον, ὁ παῖς χαίρει. | *Dōn* to dōron, ho pais chairei. | **Giving** the gift, the boy is happy. |
| **Ἐγεννώς** ὁ ἥρως τὸν πολέμῳ, ἐδόθη αὐτῷ δόξα. | *Egennōs* ho hērōs ton polemō, edothē autō doxa. | **Having become** a hero in war, he was given glory. |
| **Δούσα** τὴν ἐπιστολήν, ἡ γυνὴ ἀνέγνω αὐτήν. | *Dousa* tēn epistolēn, hē gynē anagnō autēn. | **Having given** the letter, the woman read it. |

*Note:* In narrative Greek, the aorist active participle often functions like an English **‑ing** clause (“while/after …”) or a **noun** (“the one who …ed”).

# درس: تمنایی حال میانه/مجهول فعل‌های مجهول‌نما

*سطح: میانه*
*پیش‌نیازها: فعل‌های مجهول‌نما، حال اخباری میانه/مجهول، مبانی وجه تمنایی*

---

## ۱. این صورت چیست

فعل‌های مجهول‌نما (صورت واژه‌نامه‌ای مختوم به **‑μαι**) شناسه‌های میانه/مجهول را با معنای معلوم به کار می‌برند. **تمنایی حالِ** آنها آرزو یا امکانی در جریان یا عام را می‌رساند: βουλοίμην ἄν «می‌خواستم»، εἰ δύναιτο «اگر می‌توانست».

سنگین‌وزن‌های این گروه دقیقاً همان مجهول‌نماهایی‌اند که بیش از همه به کارشان می‌برید:
**βούλομαι** «خواستن»، **δύναμαι** «توانستن»، **μάχομαι** «جنگیدن»، **ἕπομαι** «پیروی کردن»، **ἐπίσταμαι** «دانستن، بلد بودن».

## ۲. ساخت‌واژه

### ۲.۱ مجهول‌نماهای تماتیک — ‑οίμην، ‑οιο، ‑οιτο

**βούλομαι «خواستن»**:

| شخص | صورت | معنی |
|---|---|---|
| اول مفرد | **βουλοίμην** | کاش بخواهم / می‌خواستم |
| دوم مفرد | **βούλοιο** | |
| سوم مفرد | **βούλοιτο** | |
| اول جمع | **βουλοίμεθα** | |
| دوم جمع | **βούλοισθε** | |
| سوم جمع | **βούλοιντο** | |

همین الگو: μάχομαι ← **μαχοίμην، μάχοιτο**؛ ἕπομαι ← **ἑποίμην، ἕποιτο**؛ γίγνομαι (ستاک حال) ← **γιγνοίμην، γίγνοιτο**.

### ۲.۲ مجهول‌نماهای بی‌میانجی — ‑αίμην، ‑αιο، ‑αιτο

مجهول‌نماهایی که ستاکشان به ‑α ختم می‌شود (δύναμαι، ἐπίσταμαι، πρίαμαι) پسوند تمنایی را مستقیم به واکهٔ ستاک می‌چسبانند:

**δύναμαι «توانستن»**:

| شخص | صورت |
|---|---|
| اول مفرد | **δυναίμην** |
| دوم مفرد | **δύναιο** |
| سوم مفرد | **δύναιτο** |
| اول جمع | **δυναίμεθα** |
| دوم جمع | **δύναισθε** |
| سوم جمع | **δύναιντο** |

همین الگو: ἐπίσταμαι ← **ἐπισταίμην، ἐπίσταιτο**؛ πρίαμαι «خریدن» ← **πριαίμην** (از نظر فنی گذشتهٔ تام است و همچون تمناییِ ὠνέομαι به کار می‌رود).

## ۳. قاعده‌های کلیدی و استثناهای رایج

| قاعده | توضیح | نمونه |
|---|---|---|
| **۱. تماتیک‌ها ‑οι‑ می‌گیرند، بی‌میانجی‌ها ‑αι‑** | βουλοίμην اما δυναίμην. اگر اخباری βούλομαι باشد (با ‑ο‑)، تمنایی ‑οι‑ دارد؛ اگر δύναμαι باشد (بی‌واکهٔ میانجی)، ‑αι‑. | |
| **۲. دوم‌شخص مفرد σ را می‌اندازد** | βούλοι(σ)ο ← βούλοιο؛ δύναι(σ)ο ← δύναιο. | |
| **۳. یک صورت، میانه یا مجهول** | در مجهول‌نماها این پرسش پیش نمی‌آید — معنا معلوم است. در نیمه‌مجهول‌نماها بافت تصمیم می‌گیرد. | |
| **۴. سوم‌شخص جمعِ حماسی/یونیایی ‑οίατο / ‑αίατο** | هومر و هرودوت: μαχεοίατο، δυναίατο. | |
| **۵. تکیه تا آنجا که ممکن است بر دوواکهٔ تمنایی می‌ماند** | δυναίμην، ἐπισταίμην — ‑αι پایانیِ تمنایی برای تکیه بلند شمرده می‌شود. | |

## ۴. جمله‌های نمونه

- βουλοίμην ἂν ὑμῖν φίλος εἶναι. — «خوش می‌داشتم که دوست شما باشم.» (امکانیِ مؤدبانه)
- εἰ δύναιτο, βοηθήσειεν ἄν. — «اگر می‌توانست، یاری می‌کرد.»
- ἤρετο εἰ ἕποιντο οἱ σύμμαχοι. — «پرسید که آیا هم‌پیمانان در پی می‌آیند.» (تمنایی در پرسش غیرمستقیم، توالی گذشته)
- οὐκ ἂν μαχοίμην τοῖς θεοῖς. — «با خدایان نمی‌جنگیدم.»
- εἴθε ἐπίσταιο ὅσα κακὰ ποιεῖς. — «کاش می‌دانستی چه اندازه بدی می‌کنی!»

## ۵. کاربرد و بافت

۱. **βουλοίμην ἄν** تعبیر ادبِ معیار در آتیکی است — «خوش می‌دارم که...» — نرم‌تر از βούλομαι. در گفت‌وگوهای افلاطون بسیار پرکاربرد است.
۲. **کنش عام/تکرارشونده در گذشته:** εἰ (ὁπότε) + تمنایی در بند وابسته با گذشتهٔ استمراری در بند اصلی: ὁπότε μάχεσθαι δύναιντο, ἐμάχοντο «هر گاه می‌توانستند بجنگند، می‌جنگیدند».
۳. **نقل‌قول غیرمستقیم در توالی گذشته:** اخباری‌های حالِ گفتار اصلی به تمنایی حال بدل می‌شوند.
۴. تمنایی **حال** (در جریان: «توانستن، پیوسته جنگیدن») را از تمنایی **گذشتهٔ تام** همان فعل‌ها (کنش یک‌باره: γένοιτο، μαχέσαιτο) جدا کنید.

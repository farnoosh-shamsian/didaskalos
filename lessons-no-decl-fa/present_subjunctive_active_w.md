# درس: التزامی حال معلوم (فعل‌های ‑ω)

*سطح: مبتدی تا میانه*
*پیش‌نیازها: الفبای یونانی، صرف پایهٔ اسم، شناسه‌های حال اخباری معلوم.*

---

## ۱. التزامی حال معلوم چیست؟

**وجه التزامی** کنشی *بالقوه* یا *ناواقعی* را بیان می‌کند: هدف، شرط، ترغیب، یا بندهای نامعین.
**التزامیِ حال** نمودِ در جریان دارد — کنشی پیوسته یا مکرر، همزمان با فعل اصلی یا پس از آن.
وقتی فعل **معلوم** است، فاعلْ خودِ کنش را انجام می‌دهد.

نشانهٔ التزامی در یونانی **بلند شدن واکهٔ میانجی** است: ο/ε جای خود را به **ω/η** می‌دهد.

| شخص | شناسه (باقاعده) |
|--------|------------------|
| اول مفرد | ‑ω |
| دوم مفرد | ‑ῃς |
| سوم مفرد | ‑ῃ |
| اول جمع | ‑ωμεν |
| دوم جمع | ‑ητε |
| سوم جمع | ‑ωσι(ν) |

این شناسه‌ها به **ستاک حال** (همان ستاک حال اخباری) افزوده می‌شوند.

> **نکتهٔ کلیدی:** التزامی حال **افزونه نمی‌گیرد** (آن «ἐ‑» که در زمان‌های گذشته می‌آید). مانند حال اخباری است، اما با واکهٔ بلند: اخباری λύομεν ← التزامی λύωμεν.

---

## ۲. صورت‌ها و الگوها

### ۲.۱ فعل‌های تماتیک باقاعده — *λύω* «گشودن»

| | مفرد | جمع |
|----------|----------|--------|
| **اول** | λύ**ω** | λύ**ωμεν** |
| **دوم** | λύ**ῃς** | λύ**ητε** |
| **سوم** | λύ**ῃ** | λύ**ωσι(ν)** |

*نکته:* اول‌شخص مفردِ التزامی با اخباری هم‌شکل است (λύω)؛ بافت جمله تعیین‌کننده است.

### ۲.۲ فعل‌های ادغامی — *φιλέω* «دوست داشتن»

واکهٔ ستاک با واکهٔ بلندِ شناسه ادغام می‌شود (ε + ω ← ω؛ ε + ῃ ← ῇ):

| شخص | صورت | توضیح |
|--------|------|-------------|
| اول مفرد | **φιλῶ** | φιλε‑ + ‑ω ← φιλῶ |
| دوم مفرد | **φιλῇς** | φιλε‑ + ‑ῃς ← φιλῇς |
| سوم مفرد | **φιλῇ** | φιλε‑ + ‑ῃ ← φιλῇ |
| اول جمع | **φιλῶμεν** | φιλε‑ + ‑ωμεν ← φιλῶμεν |
| دوم جمع | **φιλῆτε** | φιλε‑ + ‑ητε ← φιλῆτε |
| سوم جمع | **φιλῶσι(ν)** | φιλε‑ + ‑ωσι(ν) ← φιλῶσι(ν) |

در ادغام α (τιμάω) همهٔ صورت‌ها با ᾳ/ω ساخته می‌شوند: τιμῶ, τιμᾷς, τιμᾷ, τιμῶμεν, τιμᾶτε, τιμῶσι(ν) — هم‌شکل با اخباری؛ باز هم بافت تصمیم می‌گیرد.

### ۲.۳ فعل‌های ‑μι — *δίδωμι* «دادن»

در التزامی، فعل‌های ‑μι نیز واکهٔ بلندِ التزامی را می‌گیرند و با ستاکِ دوگان‌دار ساخته می‌شوند:

| شخص | حال التزامی | توجه |
|--------|------|------|
| اول مفرد | **διδῶ** | صورت بی‌دوگان δῶ **آئوریست** است، نه حال! |
| دوم مفرد | **διδῷς** | |
| سوم مفرد | **διδῷ** | |
| اول جمع | **διδῶμεν** | |
| دوم جمع | **διδῶτε** | |
| سوم جمع | **διδῶσι(ν)** | |

### ۲.۴ فعل بی‌قاعدهٔ **εἰμί** «بودن»

التزامی حالِ εἰμί خودِ شناسه‌های بلند است:

| شخص | صورت |
|--------|------|
| اول مفرد | **ὦ** |
| دوم مفرد | **ᾖς** |
| سوم مفرد | **ᾖ** |
| اول جمع | **ὦμεν** |
| دوم جمع | **ἦτε** |
| سوم جمع | **ὦσι(ν)** |

## ۳. قاعده‌های کلیدی و استثناهای رایج

| قاعده | توضیح |
|------|-------------|
| **گزینش ستاک** | از **ستاک حال اخباری** استفاده کنید (بخش پیش از ‑ω). |
| **بدون افزونه** | التزامی حال هرگز افزونهٔ ἐ‑ نمی‌گیرد. |
| **نشانهٔ اصلی: واکهٔ بلند** | ω/η به‌جای ο/ε: λύομεν (اخباری) در برابر λύωμεν (التزامی). |
| **فعل‌های ادغامی** | ادغام مانند اخباری اما با واکه‌های بلند: φιλῶμεν، φιλῆτε. |
| **فعل‌های ‑μι** | حالِ التزامی دوگان‌سازی را نگه می‌دارد: διδῶ (حال) ≠ δῶ (آئوریست). |
| **بی‌قاعده‌ها** | εἰμί صورت‌های ویژه دارد (§۲.۴). |
| **سوم‌شخص جمع** | ‑ωσι یا ‑ωσιν؛ ν متحرک است. |
| **یوتای زیرنویس** | در ‑ῃς و ‑ῃ همیشه نوشته می‌شود و نشانهٔ مطمئن التزامی است. |

## ۴. جمله‌های نمونه

| یونانی (التزامی حال) | حرف‌نویسی | فارسی |
|-----------------------------|----------------|---------|
| **ἵνα λύῃς τὸν δεσμόν** | hína lýēis ton desmón | «تا بند را بگشایی» |
| **μανθάνομεν ἵνα σοφοὶ ὦμεν.** | manthánomen hína sophoì ômen | «می‌آموزیم تا دانا باشیم.» |
| **ἐὰν ζητῇς, εὑρήσεις.** | eàn zētêis, heurḗseis | «اگر بجویی، خواهی یافت.» |
| **γράφωμεν.** | gráphōmen | «بنویسیم!» (التزامی ترغیبی) |
| **τί ποιῶμεν;** | tí poiômen? | «چه کنیم؟» (پرسش تدبیری) |

*به کاربرد **ἵνα** (هدف) و **ἐάν** (شرط) توجه کنید که بندهای التزامی را می‌آغازند.*

---

## ۵. کاربرد و بافت

| بافت | واژه‌ها/ساخت‌های راه‌انداز |
|---------|-----------------------------------|
| **هدف** | *ἵνα، ὅπως* + التزامی: *ἵνα γράφῃς* «تا بنویسی» |
| **بندهای نامعین** | ضمیر/قید موصولی + **ἄν** + التزامی: *ὃς ἂν λέγῃ* «هر که بگوید»، *ὅταν* «هر گاه» |
| **شرط آیندهٔ محتمل** | *ἐάν* + التزامی: *ἐὰν ἔλθῃς* «اگر بیایی» |
| **ترغیبی** | اول‌شخص جمع بدون ادات: *γράφωμεν* «بنویسیم» |
| **پرسش تدبیری** | اول‌شخص: *τί ποιῶμεν;* «چه کنیم؟» |
| **بندهای ترس** | *φοβοῦμαι μή...* «می‌ترسم که مبادا...» |

التزامی برای گزارهٔ سادهٔ واقعی به کار **نمی‌رود**؛ آن قلمروِ اخباری است. التزامی همیشه رنگی از *امکان*، *خواست* یا *نامعین بودن* دارد.

## ۶. پیشینهٔ تاریخی و ریشه‌شناسی

| جنبه | جزئیات |
|--------|---------|
| **خاستگاه هندواروپایی** | التزامیِ هندواروپایی با **واکهٔ میانجی بلند** ساخته می‌شد؛ یونانی همین را یکراست ادامه می‌دهد (ω/η در برابر ο/ε). |
| **تحول در یونانی** | در هومر التزامی گاه معنای آینده‌مانند دارد؛ در دورهٔ کلاسیک وجهِ اصلیِ هدف، شرط محتمل و ترغیب است. |
| **نسبت با وجه‌های دیگر** | التزامی (واکهٔ بلند) و تمنایی (پسوند ‑οι‑/‑ιη‑) دو وجهِ «ناواقعی» یونانی‌اند؛ با پس رفتن تمنایی در کوینه، التزامی جای آن را هم گرفت. |
| **سرنوشت متأخر** | در یونانی نو، ساخت‌های na + التزامی (از ἵνα) جانشین مصدر شدند — میراث مستقیم همین وجه. |

## ۷. جدول مرور سریع (فعل تماتیک باقاعده)

| شخص | شناسه | نمونه با **λύω** |
|--------|--------|----------------------|
| اول مفرد | ‑ω | λύ**ω** |
| دوم مفرد | ‑ῃς | λύ**ῃς** |
| سوم مفرد | ‑ῃ | λύ**ῃ** |
| اول جمع | ‑ωμεν | λύ**ωμεν** |
| دوم جمع | ‑ητε | λύ**ητε** |
| سوم جمع | ‑ωσι(ν) | λύ**ωσι(ν)** |

این جدول را هنگام برخورد با هر فعل تازه دم دست داشته باشید: ستاک **λυ‑** را با ستاک حالِ فعل جایگزین کنید، قاعده‌های ادغام را اگر لازم است اعمال کنید، و التزامی حال معلوم را دارید.

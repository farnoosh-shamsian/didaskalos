# درس: تمنایی گذشتهٔ تام میانه فعل‌های مجهول‌نما

*سطح: میانه*
*پیش‌نیازها: فعل‌های مجهول‌نما، گذشتهٔ تام اخباری میانه، تمنایی حال*

---

## ۱. این صورت چیست

**فعل‌های مجهول‌نما** (صورت واژه‌نامه‌ای مختوم به **‑μαι**) صورت معلوم ندارند: صورت‌های میانه‌شان معنای معلوم دارند. پس **تمنایی گذشتهٔ تام میانهٔ** یک مجهول‌نما آرزو یا امکانی را دربارهٔ کنشی یک‌باره و کامل بیان می‌کند: γένοιτο «کاش چنین شود!»

این صورتی پرارزش است: تک‌واژهٔ **γένοιτο** (از γίγνομαι «شدن، رخ دادن») از نامدارترین تمنایی‌های یونانی است — در تعبیر μὴ γένοιτο «مبادا!» زنده مانده و پایان‌بخشِ دعاهای بی‌شمار است.

## ۲. ساخت‌واژه

بسته به گونهٔ گذشتهٔ تامِ فعل، دو ساخت وجود دارد:

### ۲.۱ مجهول‌نماهای گذشتهٔ تام دوم (تماتیک) — شناسه‌های ‑οίμην، ‑οιο، ‑οιτο

**γίγνομαι «شدن، رخ دادن»** (ستاک گذشتهٔ تام γεν‑):

| شخص | صورت | معنی |
|---|---|---|
| اول مفرد | **γενοίμην** | کاش بشوم |
| دوم مفرد | **γένοιο** | کاش بشوی |
| سوم مفرد | **γένοιτο** | کاش رخ دهد |
| اول جمع | **γενοίμεθα** | کاش بشویم |
| دوم جمع | **γένοισθε** | کاش بشوید |
| سوم جمع | **γένοιντο** | کاش بشوند |

همین الگو: πυνθάνομαι «با پرس‌وجو دانستن» ← **πυθοίμην، πύθοιο، πύθοιτο**؛ ἀφικνέομαι «رسیدن» ← **ἀφικοίμην، ἀφίκοιτο**؛ (شاعرانه) ἱκνέομαι «آمدن، رسیدن» ← **ἱκοίμην، ἵκοιτο**.

### ۲.۲ مجهول‌نماهای گذشتهٔ تام اول (سیگمایی) — شناسه‌های ‑αίμην، ‑αιο، ‑αιτο

**ἡγέομαι «رهبری کردن؛ پنداشتن»** (گذشتهٔ تام ἡγησάμην):

| شخص | صورت |
|---|---|
| اول مفرد | **ἡγησαίμην** |
| دوم مفرد | **ἡγήσαιο** |
| سوم مفرد | **ἡγήσαιτο** |
| اول جمع | **ἡγησαίμεθα** |
| دوم جمع | **ἡγήσαισθε** |
| سوم جمع | **ἡγήσαιντο** |

همین الگو: δέχομαι «پذیرفتن» ← **δεξαίμην، δέξαιτο**؛ αἰσθάνομαι «دریافتن» ← **αἰσθοίμην** (گذشتهٔ تام دوم) اما θεάομαι «نگریستن» ← **θεασαίμην** (گذشتهٔ تام اول).

## ۳. قاعده‌های کلیدی و استثناهای رایج

| قاعده | توضیح | نمونه |
|---|---|---|
| **۱. ستاک گذشتهٔ تامِ فعل را بدانید** | γίγνομαι گذشتهٔ تام دومِ ἐγενόμην دارد ← γενοίμην؛ ἡγέομαι گذشتهٔ تام اولِ ἡγησάμην دارد ← ἡγησαίμην. تمنایی از گونهٔ ستاکِ اخباری پیروی می‌کند. | |
| **۲. بدون افزونه** | افزونه تنها از آنِ اخباری است: ἐγένετο اما γένοιτο. | |
| **۳. دوم‌شخص مفرد به ‑οιο / ‑αιο ختم می‌شود** | از ‑οι‑σο با افتادن σ: γένοιο «کاش بشوی». | ὄλβιος γένοιο |
| **۴. سوم‌شخص جمعِ حماسی/یونیایی ‑οίατο** | نزد هومر و هرودوت، γενοίατο = γένοιντο. | |
| **۵. اول‌شخص مفردِ دوری/شاعرانه ‑οίμαν** | در سرود همسرایان و تراژدی: γενοίμαν. | (سوفوکلس، همسرایان) |

## ۴. جمله‌های نمونه

- μὴ γένοιτο. — «مبادا چنین شود!» / «خدا نکند!»
- ὄλβιος γένοιο, φίλε. — «کاش خجسته باشی، ای دوست.»
- εἰ τοῦτο γένοιτο, πάντες ἂν σωθεῖμεν. — «اگر این رخ می‌داد، همه رهایی می‌یافتیم.» (شرط دور از انتظار)
- ἔπεμψε κατάσκοπον ὅπως πύθοιτο τὰ τῶν πολεμίων. — «دیده‌بانی فرستاد تا از کارِ دشمنان آگاه شود.» (بند هدف در توالی گذشته)
- οὐκ ἂν ἡγήσαιτο τοῦτο δίκαιον εἶναι. — «این را دادگرانه نمی‌شمرد.» (تمنایی امکانی)

## ۵. کاربرد و بافت

۱. **آرزوها:** تمنایی برهنه (اغلب با εἴθε یا εἰ γάρ) آرزویی برای آینده را می‌رساند: γένοιτο «چنین باد» — «آمینِ» یونانی.
۲. **بندهای هدف و ترس در توالی گذشته:** پس از فعل اصلیِ گذشته، ἵνα/ὅπως/μή تمنایی می‌گیرند: ἦλθεν ἵνα πύθοιτο «آمد تا آگاه شود».
۳. **شرط‌های دور از انتظار:** εἰ + تمنایی … ἄν + تمنایی: «اگر X رخ می‌داد، Y رخ می‌داد».
۴. **تعبیرهای نفرین و دعا** به این صورت دلبسته‌اند — مجهول‌نماهای شدن، پذیرفتن و دریافتن در وجه تمنایی، ستون فقرات دعاهای یونانی‌اند.
